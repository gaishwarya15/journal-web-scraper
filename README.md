This project focuses on web scraping academic metadata from the current issue of the *Journal of Marketing*. It automates the extraction of:

- Article Titles
- Author Names and Affiliations
- Abstracts and DOIs
- First Published Dates

The scraper uses `Python`, `Selenium` and `Chrome WebDriver` to collect and structure data into a clean CSV file. It includes:

- Dynamic handling of article sections
- Error-tolerant scraping logic for missing fields
- Clean data output suitable for research use
